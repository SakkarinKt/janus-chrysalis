## Claims

- **[high, self_checked — near-verbatim abstract]** MMSA unifies joint state-action representation learning with imaginative rollouts: a world model trained with a variational autoencoder is augmented with a "state-action learned embedding" (SALE), injected into (a) the imagination module that forecasts future rollouts and (b) a joint-agent network whose per-agent action values are combined via a mixing network into a joint action-value function.
- **[high, self_checked]** This is a value-decomposition-style architecture (QMIX-lineage mixing network) fused with a world model — a different axis than the shared/per-agent-world-model spectrum the rest of the lit map is organized around. MMSA's world model itself is not described (so far) as either fully shared or fully per-agent-with-explicit-communication; the representation-sharing happens at the *action-value* level (via the mixing network) rather than at the *dynamics-model* level. Flagged as a possible 6th, orthogonal axis point: "shared value decomposition, independent world models" — needs primary-source confirmation of whether the world model itself is per-agent or joint before placing it definitively on the lit-map's axis.
- **[medium, self_checked]** No explicit non-stationarity framing surfaced in secondary sources for this paper (unlike MARIE, GAWM, MATWM) — its stated motivation is data efficiency and joint-outcome understanding via richer state-action embeddings, not co-learning non-stationarity specifically. If confirmed on a primary read, this would make it the second paper in the lit map (after the original MAMBA/MABL/CoDreamer batch) that doesn't name non-stationarity as a first-class target — relevant context for how central that framing actually is across the field vs. how central it looked from the first four papers alone.
- **[low, self_checked]** Evaluated on Multi-Agent MuJoCo, StarCraft II (SMAC), and Level-Based Foraging (LBF) — from secondary summary; benchmark list not yet cross-checked against the paper's own experiments section.

## Method summary (best available from abstract + secondary sources)

A VAE-based world model learns joint state-action embeddings (SALE) shared across the imagination and value-estimation pathways. During imagination, SALE-conditioned rollouts are generated; the resulting imagined trajectories feed a joint-agent value network whose individual agents' action-values are combined through a mixing network (QMIX-style) into a single joint action-value function used for policy improvement. The stated benefit is that agents get a "richer understanding of how their choices influence collective outcomes" than value decomposition alone would give, because the value function is trained partly on model-generated (imagined) trajectories informed by the joint embedding.
